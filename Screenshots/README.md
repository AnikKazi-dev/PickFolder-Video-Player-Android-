# Screenshots

Add your app screenshots here:

1. **home.png** - Main screen showing the folder selection button and video list
2. **folder_selection.png** - Android's folder picker interface
3. **video_player.png** - Video playing with controls visible
4. **controls.png** - Video player controls (play/pause, seek, next/previous)
5. **app_icon.png** - App icon (optional)

## How to Capture Screenshots

1. Run the app on your Android device
2. Use your device's screenshot feature (usually Power + Volume Down)
3. Save screenshots to this folder
4. Name them according to the list above

## Recommended Screenshot Sizes

- Phone: 1080x2340 or similar (9:19.5 aspect ratio)
- Keep file sizes under 1MB each
