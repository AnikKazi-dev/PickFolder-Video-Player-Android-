# PickFolder Player v1.0.0

## Initial Release

Privacy-focused video player for Android with specific folder selection.

### Features

- **Secure Folder Selection**: Select specific folders using Android's Storage Access Framework
- **Shuffle Mode**: Randomize video playback order
- **Autoplay**: Automatically play the next video
- **Full-Featured Player**: Play/Pause, Seek, Previous/Next controls
- **Persistent Settings**: Remembers folder, shuffle, and autoplay preferences
- **Dark Theme**: Modern Material Design 3 interface

### Supported Video Formats

MP4, MKV, AVI, MOV, WMV, FLV, WebM, M4V, 3GP, MPEG, MPG

### Requirements

- Android 5.0 (API level 21) or higher

### Installation

1. Download the APK file below
2. Enable "Install from Unknown Sources" in your Android settings
3. Install the APK
4. Grant necessary permissions when prompted
5. Enjoy your videos with privacy!

### Privacy & Security

Unlike other video players that require full storage access, PickFolder Player only accesses the specific folders you select, keeping your private files secure.
